# Down-scaled and processed excitation emission matrices

Processed version of
[`example_eems()`](https://katiewampler.github.io/eemanalyzeR/reference/example_eems.md)
using `eemanalyzeR`.

## Usage

``` r
example_processed_eems
```

## Format

An `eemlist` with four samples:

- **B1S1ExampleBlankSEM**: Fluorescence measured for a sample of Type 1
  water

- **B1S2ExampleTeaStdSEM**: Fluorescence measured for a sample of dilute
  unsweetened tea

- **B1S3ExampleSampleSEM**: Fluorescence measured for a sample of stream
  water from Oregon, USA

- **ManualExampleTeaWaterfallPlotSample**: Fluorescence measured for a
  sample of dilute unsweetened tea
